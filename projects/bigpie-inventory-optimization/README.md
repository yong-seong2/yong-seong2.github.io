# 수요급등 대비 안전재고량 최적화

**AI융합 산학연계 해커톤** · 팀 빅파이(4인) · 팀 리더·모델링·발표 담당 · 2025.08

음료 상품의 월별 판매 데이터를 기반으로 수요를 예측하고, 온도·기념일 시나리오에 따른 안전재고 최적화 방안을 제시한 프로젝트입니다.

## 결과
- WAPE 19.66% — 리더보드 1위

## 분석 구조
- 품목별 판매 규모(월평균 판매량, 데이터 보유 기간)를 기준으로 모델을 이원화
  - 고판매량: CatBoost (lag/rolling mean 피처)
  - 저판매량: SARIMAX (계절성 + 쿠폰·프로모션 외생변수, Fallback 로직 포함)
- 예측치-직전월 실측치를 변동계수(CV) 기반 alpha로 블렌딩해 노이즈 완화
- 직전 6개월 평균 기반 스파이크 캡으로 비현실적 급등 예측 방지
- 시나리오1: 온도 +5°C 상승 시 판매량 변화 분석
- 시나리오2: 기념일 전후 재구매율 변화 분석
